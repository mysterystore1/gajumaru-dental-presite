# 辻堂がじゅまる歯科 装飾イラスト スタイルガイド v1

全SVG装飾素材はこのガイドに準拠する。新規作成・修正時はこのファイルを必ず参照。

## 1. パレット（4色＋不透明度3段階のみ）

| 役割 | 値 | CSS変数（新サイト移行後） |
|---|---|---|
| 線・点目 | #9C5026 | var(--illust-line) |
| 塗り・主 | #FBF6F0 | var(--illust-fill) |
| 塗り・副（葉・くちばし等） | #F7E8DB | var(--illust-fill-sub) |
| 頬・内耳 | #F0DAC8 | var(--illust-blush) |

- 不透明度は **0.45 / 0.5 / 0.6** の3段階のみ（背景装飾の遠近表現用）
- キャラクター本体（猫・鳥）は常に opacity 1
- 上記以外の色・不透明度の新設は禁止

## 2. 線のルール（3段階のみ）

| 用途 | stroke-width |
|---|---|
| 主線（体・頭・耳の輪郭） | 2.4〜2.5 |
| 細部線（羽・足・内側のライン・小さな葉） | 1.4〜1.8 |
| 極細（葉脈・ひげ） | 0.9 |

- stroke-linecap: round / stroke-linejoin: round を全パスで固定
- しっぽ等の太い要素は「太いストローク」ではなく **テーパー付きの閉じた輪郭パス**（塗り＋主線）で描く

## 3. 作画ルール

1. **真円・直線の禁止**：頭は真円（circle要素）でなく4ベジェの有機的な閉パス。耳は直線三角でなく、外辺わずかに凸・内辺わずかに凹の曲線三角
2. **非対称の原則**：左右の耳・葉の傾きは意図的に非対称にする（左右コピーの反転使用禁止）
3. **顔の文法**：点目（r=1.6の円、線色）＋ω口（線幅1.3）＋頬（rx2.4/ry1.6の楕円、頬色）＋ひげ（極細・opacity 0.5・片側2本）
4. **内耳**：頬色の小さな閉パスを必ず入れる
5. **描画順**：しっぽ→体→耳→内耳→頭→顔パーツ（接合部を上のレイヤーの線で隠す）
6. **葉の文法**：2ベジェの雫型＋中央葉脈（0.9）。舞う葉はrotate＋opacity 0.55〜0.6

## 4. 猫キャラクター仕様（全ポーズ共通）

- viewBox座標系: 90×80 基準（cat-yawn / cat-sitting と同一）
- 頭身: 頭の縦幅 ≒ 体の縦幅 × 0.55（子猫プロポーション維持）
- 頭中心: (42, 26)付近 / 半径換算 10.5〜11.5
- しっぽ: 体右側から出て上向きにひと巻き。基部太・先端細のテーパー
- 表情バリエーションは目と口のみ変更（点目=通常 / 閉じ目カーブ=あくび・眠り）。輪郭は共通

## 5. 配置ルール（レビュー指摘の反映）

- 装飾は1セクション1点まで
- キャラクター（猫・鳥）は感情接点のあるセクション限定：ヒーロー / 恐怖層向け / FAQ / 予約CTA / フッター
- 背景装飾（葉・気根・足あと）はopacity 0.45〜0.6、サイズ32〜72px相当
- 実装時は色をCSS変数参照に置換し、トークンと連動させる

## 6. ファイル規約

- ファイル名: deco-{モチーフ}-{ポーズ}.svg（例: deco-cat-sitting.svg）
- 各ファイル冒頭にスタイル準拠コメントを残す
- 1ファイル2KB以下を目安
