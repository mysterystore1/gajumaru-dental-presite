## v2追記（2026-06-10）: 競合実院との「レベル差」分析→院長肉声追加
比較対象: shonan-cx-dc.com/clinic/ ・ tsujido-kawashima-dental.com/about/（共に辻堂・湘南）

### 学び: AI感の正体は「言葉選び」より「具体的な現実の不在」
実院が"人が書いた"感を出す装置（ウチに不足）:
1. **実写真を来院導線順に**（外観→入口→待合→診療室→機器＋機能説明）← **最大の差。実写真待ち（内覧会後 約2週間）。揃えばレベルが最も上がる**
2. 院長個人の肉声・モットー（「"自分がされて嫌なことは患者にしない"をモットーに」）
3. 具体の数字＋慎重表現（「初診60分程度」「90〜120日で形成されるといわれています」）
4. 患者心理の先回り＋感情語（「これはとても悲しいことです」）
5. 読み手への問いかけ（「〜聞いたことはありませんか?」）

### v2で実施（仮の範囲・捏造なし）
- **院長メッセージブロック追加**（Q&Aとページ CTA の間）。声は index のごあいさつ（桑田淳院長）と一貫。問いかけ＋感情語（もったいない）＋具体（歯科が苦手な方は20人に1人ほどともいわれています・仮）＋モットー的姿勢。捏造エピソード（私自身苦手だった等）は入れない＝index人格踏襲
- CSS `.doctor-message` 追加（白カード＋左アクセント罫＋署名右寄せ）

### 残（レベルを上げる最重要・今は不可）
- **実院内写真の差し替え**（内覧会後）＝①。これが最大テコ
- 実院長エピソード・実データ・実NAP（確定後・誤情報をMEOに出さない原則）

## 地元競合ベンチマーク（2026-06-10・辻堂/湘南/藤沢の6院）
湘南C-X / 辻堂かわしま / 辻堂カバサワ / 湘南辻堂駅前 / 藤沢かわい / 辻堂ステラ

### 横断して共通する勝ちパターン
1. **「かかりつけ・家族で通える」positioning が一貫**（「地域に寄り添うかかりつけ歯科医」「家族皆さまの健康を長く支えていける歯科医院でありたい」）→ ウチの index「家族で通いやすい歯科医院を目指して」と方向一致。anxiety もこの軸に接続済
2. **院長の顔写真＋一人称ごあいさつが必ずある**（藤沢かわい「私は皆さまとの会話を大切にし…お気軽に遊びにいらしてください」/ ステラ 院長「ようこそ」）→ **院長の実顔写真が最大の信頼装置。今は未取得**
3. **「〜でありたい」理想像表現**（断定を避け、姿勢で語る）→ 採用済
4. **初診はカウンセリング・検査のみの丁寧な前置き**（藤沢かわい「初診時は急性の痛みを除きカウンセリングと検査のみとなることを、どうかご理解いただけますと幸いです」）→ ウチも実装済
5. **五感・空間描写**（「木のぬくもり、アロマ香る安らぎ空間」）→ 実院内写真とセットで効く＝写真待ち
6. **具体スケジュール・実データ**（「第2・第4木曜 ママさん管理栄養士セミナー」）→ 確定後。捏造不可

### 結論（正直）
- **声（コピー）は地元競合と互角の水準に到達**。語り口・寄り添い・前置きの丁寧さは6院と同等
- **残るレベル差の約9割は「実アセット」**＝院長顔写真・院内実写・院長経歴・実NAP/実データ。これらは（仮）解消＝内覧会後/確定後にしか埋まらない
- → コピーのさらなる微調整は逓減。**次にレベルを最も上げるのは実写真の差し替え**（最優先で待機）
